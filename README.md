# VerityArc (Web)

## Requirements
- Node.js (LTS)
- npm
- Supabase project + env vars

## Setup
1. Install dependencies:
   npm install

2. Create .env.local in the repo root with:
   NEXT_PUBLIC_SUPABASE_URL=...
   NEXT_PUBLIC_SUPABASE_ANON_KEY=...

3. Run dev server:
   npm run dev

Open http://localhost:3000
